Fundamental Theorem of Calculus

Complete Exercise Solutions - Parts 1 & 2

Part 1: Finding Derivatives Using FTC Part 1

Use Part 1 of the Fundamental Theorem of Calculus to find the derivative of the function.
FTC Part 1 - Mga Formula na Kailangan Natin:

Basic Form: Kung g(x) = ∫xa f(t) dt, then g'(x) = f(x)
Ibig sabihin: Ang derivative ng integral ay yung function sa loob, pero x na ang variable.

Extended Form: Kung g(x) = ∫u(x)a f(t) dt, then g'(x) = f(u(x)) · u'(x)
Ibig sabihin: Pag may function sa upper limit (hindi plain x), kailangan ng Chain Rule.
1. g(x) = ∫x0 √(1 + 2t) dt
Given: g(x) = ∫x0 √(1 + 2t) dt
Unang tanong natin: Anong type ito ng problem? Tingnan natin ang upper limit - plain x lang siya, hindi function ng x. So basic FTC Part 1 lang gagamitin natin.
Step 1: I-identify ang form - ito ay ∫xa f(t) dt
• a = 0 (lower limit)
• f(t) = √(1 + 2t) (function sa loob ng integral)
Why ganito? Kasi ang FTC Part 1 basic form ay ginagamit kapag ang upper limit ay plain x lang. Walang complication dito.
Step 2: Apply ang FTC Part 1 basic formula
Since g(x) = ∫x0 f(t) dt, therefore g'(x) = f(x)
Step 3: I-substitute ang x sa function f(t)
f(t) = √(1 + 2t)
f(x) = √(1 + 2x) (palitan lang ang t ng x)
Bakit ganito? Ang FTC Part 1 ay nagsasabi na ang derivative ng integral ay yung original function sa loob, pero ang variable ay yung upper limit (which is x sa case natin).
g'(x) = √(1 + 2x)
Tip: Pag nakita mo na ang upper limit ay plain x, automatic na basic FTC Part 1 lang gagamitin mo. Hindi na kailangan ng Chain Rule.
2. g(x) = ∫x1 ln t dt
Given: g(x) = ∫x1 ln t dt
Analysis: Tignan natin - upper limit ay x, lower limit ay 1, function sa loob ay ln t. Same pattern din ito sa problem 1, so basic FTC Part 1 lang ulit.
Step 1: I-identify ang components
• a = 1 (lower limit)
• f(t) = ln t (function sa loob)
Step 2: Apply ang basic FTC Part 1
g'(x) = f(x) (direktang application lang)
Step 3: I-substitute ang x
f(t) = ln t → f(x) = ln x
Super straightforward ito! Walang complication kasi plain x ang upper limit. Ang derivative ng ∫₁ˣ ln t dt ay ln x lang.
g'(x) = ln x
3. y = ∫cos x1 (t + sin t) dt
Given: y = ∫cos x1 (t + sin t) dt
Ito ang challenging part! Notice na ang upper limit ay cos x, hindi plain x. Ibig sabihin may function sa upper limit, kaya kailangan natin ng extended form ng FTC Part 1 plus Chain Rule.
Step 1: I-identify na ito ay extended form
• a = 1 (lower limit)
• u(x) = cos x (upper limit - function of x!)
• f(t) = t + sin t (function sa loob)
Key insight: Pag may function sa upper limit instead of plain x, automatic na extended form plus Chain Rule ang gagamitin natin.
Step 2: Kumuha ng derivative ng upper limit function
u(x) = cos x
u'(x) = -sin x
Step 3: I-evaluate ang f(u(x))
f(t) = t + sin t
f(u(x)) = f(cos x) = cos x + sin(cos x)
Explanation: Dito pinapalitan natin ang t ng cos x sa function f(t). So t + sin t becomes cos x + sin(cos x).
Step 4: Apply ang extended FTC Part 1 formula
dy/dx = f(u(x)) · u'(x)
dy/dx = [cos x + sin(cos x)] · (-sin x)
dy/dx = -sin x[cos x + sin(cos x)]
Chain Rule in action: Parang nag-differentiate tayo ng composite function. Yung f(u(x)) times u'(x) - classic Chain Rule pattern.
dy/dx = -sin x(cos x + sin(cos x))
Remember: Pag may function sa upper limit (like cos x, e^x, x², etc.), always use extended form plus Chain Rule. Hindi lang plain substitution.

Part 2: Evaluating Definite Integrals Using FTC Part 2

Use Part 2 of the Fundamental Theorem of Calculus to evaluate the integral, or explain why it does not exist.
FTC Part 2 - Ang Main Formula:

Kung ang f ay continuous sa [a,b] at F ay antiderivative ng f, then:
∫ba f(x) dx = F(b) - F(a)

Simple process: 1) Find antiderivative, 2) Substitute upper limit, 3) Subtract lower limit result
1. ∫3-1 x5 dx
Game plan: Kailangan natin ng antiderivative ng x⁵, then i-evaluate natin from -1 to 3 using FTC Part 2.
Step 1: Find ang antiderivative ng f(x) = x5
Using power rule: ∫ xn dx = (xn+1)/(n+1) + C
F(x) = ∫ x5 dx = (x6)/6 + C
Why x⁶/6? Power rule: add 1 sa exponent (5+1=6), then divide by new exponent (÷6). Hindi na kailangan ng +C kasi definite integral tayo.
Step 2: Apply FTC Part 2 formula
∫3-1 x5 dx = F(3) - F(-1)
Step 3: I-compute ang F(3)
F(3) = (36)/6 = 729/6
Step 4: I-compute ang F(-1)
F(-1) = ((-1)6)/6 = 1/6 (kasi (-1)⁶ = 1)
Remember: (-1)⁶ = 1 kasi even exponent. Pag odd exponent, negative pa rin.
Step 5: Subtract to get final answer
∫3-1 x5 dx = 729/6 - 1/6 = 728/6 = 364/3
∫3-1 x5 dx = 364/3
2. ∫82 (4x + 3) dx
Strategy: Linear function ito, so straightforward ang antiderivative. I-integrate term by term, then evaluate.
Step 1: Find ang antiderivative ng f(x) = 4x + 3
∫ (4x + 3) dx = ∫ 4x dx + ∫ 3 dx
= 4 · (x2/2) + 3x + C
F(x) = 2x2 + 3x + C
Breakdown: ∫4x dx = 4∫x dx = 4(x²/2) = 2x², at ∫3 dx = 3x. Simple lang!
Step 2: Apply FTC Part 2
∫82 (4x + 3) dx = F(8) - F(2)
Step 3: Calculate F(8)
F(8) = 2(8)2 + 3(8) = 2(64) + 24 = 128 + 24 = 152
Step 4: Calculate F(2)
F(2) = 2(2)2 + 3(2) = 2(4) + 6 = 8 + 6 = 14
Step 5: Final computation
∫82 (4x + 3) dx = 152 - 14 = 138
∫82 (4x + 3) dx = 138
3. ∫2ππ cos θ dθ
Trig function ito: Kailangan natin alalahanin ang basic trig integrals. Ang antiderivative ng cos θ ay sin θ.
Step 1: Find ang antiderivative ng f(θ) = cos θ
F(θ) = ∫ cos θ dθ = sin θ + C
Basic trig integral: Always remember na d/dx[sin θ] = cos θ, kaya ∫cos θ dθ = sin θ.
Step 2: Apply FTC Part 2
∫2ππ cos θ dθ = F(2π) - F(π)
= sin(2π) - sin(π)
Step 3: Evaluate ang trig values
sin(2π) = 0 (full revolution sa unit circle)
sin(π) = 0 (halfway point sa unit circle)
Unit circle knowledge: Sa π at 2π, nasa x-axis tayo ng unit circle, kaya sin value ay 0 pareho.
Step 4: Final calculation
∫2ππ cos θ dθ = 0 - 0 = 0
∫2ππ cos θ dθ = 0
Interesting result: Zero ito kasi yung area above x-axis cancels out yung area below x-axis sa interval [π, 2π].
4. ∫5-2 6 dx
Constant function ito: Simplest type ng integral. Ang antiderivative ng constant ay constant times x.
Step 1: Find ang antiderivative ng constant f(x) = 6
F(x) = ∫ 6 dx = 6x + C
Constant rule: ∫k dx = kx + C, where k is any constant. Kaya ∫6 dx = 6x.
Step 2: Apply FTC Part 2
∫5-2 6 dx = F(5) - F(-2) = 6(5) - 6(-2)
Step 3: Simple arithmetic
= 30 - (-12) = 30 + 12 = 42
Geometric interpretation: Ito ay area ng rectangle na height = 6 at width = 5 - (-2) = 7. So 6 × 7 = 42. ✓
∫5-2 6 dx = 42
5. ∫π/40 sec2 t dt
Special trig integral: Kailangan natin alalahanin na ang antiderivative ng sec²t ay tan t. Ito ay common sa calculus.
Step 1: Find ang antiderivative ng f(t) = sec2 t
F(t) = ∫ sec2 t dt = tan t + C
Why tan t? Kasi ang derivative ng tan t ay sec²t. Remember: d/dx[tan t] = sec²t, so ∫sec²t dt = tan t.
Step 2: Apply FTC Part 2
∫π/40 sec2 t dt = F(π/4) - F(0)
= tan(π/4) - tan(0)
Step 3: Evaluate ang trig values
tan(π/4) = 1 (45° angle sa unit circle)
tan(0) = 0 (starting point sa unit circle)
Unit circle recall: Sa π/4 (or 45°), opposite at adjacent sides ay equal, kaya tan(π/4) = 1/1 = 1.
Step 4: Final result
∫π/40 sec2 t dt = 1 - 0 = 1
∫π/40 sec2 t dt = 1
6. Compute ∫50 f(x) dx given that f(x) = {3 for x < 3, x for x ≥ 3}
Piecewise function challenge: Hindi ito tuloy-tuloy na function, may "jump" sa x = 3. Kailangan natin i-split ang integral sa point na yan.
Step 1: Split ang integral sa breakpoint x = 3
∫50 f(x) dx = ∫30 f(x) dx + ∫53 f(x) dx
Why split? Kasi sa [0,3), f(x) = 3, pero sa [3,5], f(x) = x. Different functions sila, so separate integrals.
Step 2: Evaluate ang first integral where f(x) = 3 for x < 3
∫30 3 dx = 3x |03 = 3(3) - 3(0) = 9
Simple constant integral: From 0 to 3, constant function siya na 3. Area = height × width = 3 × 3 = 9.
Step 3: Evaluate ang second integral where f(x) = x for x ≥ 3
∫53 x dx = (x2/2) |35 = (52/2) - (32/2)
= 25/2 - 9/2 = 16/2 = 8
Power rule application: Sa [3,5], f(x) = x. Antiderivative ay x²/2. Evaluate from 3 to 5.
Step 4: Add ang dalawang results
∫50 f(x) dx = 9 + 8 = 17
∫50 f(x) dx = 17
Piecewise Strategy: Always split sa mga breakpoints ng function. Each piece ay i-integrate separately, then i-add ang results.
7. ∫2-1 |x| dx
Absolute value function: Ito rin ay piecewise! Remember na |x| = x pag x ≥ 0, at |x| = -x pag x < 0.
Step 1: I-define ang piecewise function
|x| = {-x for x < 0, x for x ≥ 0}
Step 2: Split ang integral sa x = 0 (critical point)
∫2-1 |x| dx = ∫0-1 |x| dx + ∫20 |x| dx
Step 3: Evaluate first integral where |x| = -x for x < 0
∫0-1 (-x) dx = (-x2/2) |-10
= -02/2 - (-(-1)2/2) = 0 - (-1/2) = 1/2
Careful with signs: Sa [-1,0], |x| = -x. Antiderivative ng -x ay -x²/2.
Step 4: Evaluate second integral where |x| = x for x ≥ 0
∫20 x dx = (x2/2) |02 = 22/2 - 02/2 = 4/2 - 0 = 2
Step 5: Add ang results
∫2-1 |x| dx = 1/2 + 2 = 5/2
∫2-1 |x| dx = 5/2
Geometric check: Ito ay area ng dalawang triangles: triangle sa [-1,0] (area = 1/2) + triangle sa [0,2] (area = 2). Total = 5/2. ✓
8. ∫1-1 (1/x2) dx
Improper integral alert! May discontinuity sa x = 0, na nasa loob ng interval [-1,1]. Hindi ito well-defined.
Step 1: I-check ang domain ng function
f(x) = 1/x2 is undefined at x = 0
Problem identified: Ang function ay hindi continuous sa [−1,1] kasi may vertical asymptote sa x = 0. Hindi natin pwedeng gamitin ang FTC Part 2 directly.
FTC Part 2 Requirements: Ang function dapat continuous sa closed interval [a,b]. Since may discontinuity sa x = 0, hindi applicable ang FTC Part 2.
Step 2: Check if pwedeng maging improper integral
Para maging well-defined, kailangan natin mag-limit:
∫1-1 (1/x2) dx = lima→0- ∫a-1 (1/x2) dx + limb→0+ ∫1b (1/x2) dx
Step 3: Evaluate ang limits (if they exist)
∫ (1/x2) dx = ∫ x-2 dx = -x-1 = -1/x

First limit: lima→0- [-1/x]a-1 = lima→0- (-1/a - (-1/(-1))) = lima→0- (-1/a - 1) = +∞
Second limit: limb→0+ [-1/x]1b = limb→0+ (-1/1 - (-1/b)) = limb→0+ (-1 + 1/b) = +∞
Result: Both limits diverge to infinity, so ang improper integral ay divergent.
∫1-1 (1/x2) dx does not exist (divergent improper integral)
Key Lesson: Always check if ang function ay continuous sa interval before using FTC Part 2. Pag may discontinuity, consider if improper integral siya at kung convergent ba.

9. Find d/dx[∫x32 √(t2 + 1) dt]
Advanced FTC Part 1: May x³ sa upper limit, so kailangan ng extended form plus Chain Rule.
Step 1: I-identify ang components
• u(x) = x3 (upper limit function)
• f(t) = √(t2 + 1) (integrand)
Step 2: Find u'(x)
u'(x) = 3x2
Step 3: Find f(u(x))
f(u(x)) = f(x3) = √((x3)2 + 1) = √(x6 + 1)
Step 4: Apply extended FTC Part 1
d/dx[∫x32 √(t2 + 1) dt] = f(u(x)) · u'(x)
= √(x6 + 1) · 3x2
d/dx[∫x32 √(t2 + 1) dt] = 3x2√(x6 + 1)
10. Evaluate ∫π0 x sin x dx
Integration by parts needed: Hindi ito straightforward antiderivative, kailangan ng integration by parts technique.
Step 1: Set up integration by parts: ∫u dv = uv - ∫v du
Let u = x, so du = dx
Let dv = sin x dx, so v = -cos x
Why choose u = x? Kasi mas simple ang derivative ng x (which is 1) compared sa derivative ng sin x.
Step 2: Apply ang integration by parts formula
∫ x sin x dx = x(-cos x) - ∫ (-cos x) dx
= -x cos x + ∫ cos x dx
= -x cos x + sin x + C
Step 3: Apply FTC Part 2
∫π0 x sin x dx = [-x cos x + sin x]π0
Step 4: Evaluate at limits
At x = π: -π cos(π) + sin(π) = -π(-1) + 0 = π
At x = 0: -0 cos(0) + sin(0) = 0 + 0 = 0
Step 5: Final computation
∫π0 x sin x dx = π - 0 = π
∫π0 x sin x dx = π

Summary & Key Takeaways

FTC Part 1 - Quick Reference

Basic Form: If g(x) = ∫xa f(t) dt, then g'(x) = f(x)
Use when: Upper limit is plain x

Extended Form: If g(x) = ∫u(x)a f(t) dt, then g'(x) = f(u(x)) · u'(x)
Use when: Upper limit is a function of x (like x², cos x, e^x, etc.)

Decision Process:
1. Look at upper limit
2. If plain x → use basic form
3. If function of x → use extended form + Chain Rule

FTC Part 2 - Quick Reference

Formula: ∫ba f(x) dx = F(b) - F(a) where F is any antiderivative of f

Requirements:
• f must be continuous on [a,b]
• You must find an antiderivative F

Process:
1. Find antiderivative F(x)
2. Evaluate F(b)
3. Evaluate F(a)
4. Compute F(b) - F(a)

Special Cases to Remember

Piecewise Functions: Split integral at breakpoints
Absolute Value: Split at zeros of the expression inside | |
Discontinuous Functions: Check if improper integral exists
Integration by Parts: Use for products like x·sin x, x·e^x, etc.

Common Antiderivatives:
• ∫ x^n dx = x^(n+1)/(n+1) (power rule)
• ∫ cos x dx = sin x
• ∫ sin x dx = -cos x
• ∫ sec²x dx = tan x
• ∫ 1/x dx = ln|x|
Final Study Tip: Practice identifying which part of FTC to use by looking at the problem structure first. FTC Part 1 = finding derivatives of integrals, FTC Part 2 = evaluating definite integrals. Know the difference!
